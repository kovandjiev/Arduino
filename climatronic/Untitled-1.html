The board is powered by Atmel’s SAMD21 MCU, which features a 32-bit ARM Cortex® M0+ core.

ProDino MKR Zero is a simple and powerful 32-bit board. This board aims to provide a platform for innovative projects in smart IoT devices, wearable technology, high-tech automation and much more.

 

ProDino MKR Zero has vary variants: 

  • ProDino MKR Zero Base: 
    • 4x optical isolated inputs
    • 4x outputs with relays 250V 5A 
    • 1x Grove digital connector supports vary sensors: temperature, humidity and much more  
    • 1x RS485 allows communicate with other devices
    It compatibles with "Arduino MKRZero". This variant is the most suitable for separately working device, extender on other devices connected to RS485 bus.
  • ProDino MKR Zero GSM includes all "Base" features:
    • GSM 1400 module
    It compatible Arduino MKR GSM 1400. This variant is the most suitable for device which works without Ethernet cable connection. It can work every place where has an GSM connection.
  • ProDino MKR Zero Eth includes all "Base" features:
    • Ethernet Wiznet W5500 module
    It compatibles with "Arduino MKRZero". This variant is the most suitable for IoT devices. 
  • ProDino MKR Zero Eth + GSM includes all "Base" features:
    • GSM 1400 module 
    • Ethernet Wiznet W5500 module
    It compatibles with Arduino MKR GSM 1400. This variant is the most suitable for IoT devices, bridge GSM <-> Ethernet, control Ethernet and send information through GSM connection, GPS tracker, remote GSM control.
